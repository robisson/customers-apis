# Customers and favorite producs

Open source service api to manage customers and favorite producs.

**Made with**
* Typescript
* NestJs
* REST Api
* MongoDB

**Deplyoment models**
* Docker + Kubernetes
* Docker + Kubernetes Istio Service Mesh

## How to use


#### For development

```sh
yarn

yarn start:dev
```

#### For production
Buid and run it:

```sh
yarn start
```
